System Noise Calculator

This program uses the equations described in an article by Hasan Schiers N0AN in Orbit magazine No 14. The article is “AO-10 Ground Station Analysis”.

I have placed a copy of this article in dropbox.

See: https://www.dropbox.com/sh/rvb80ajqcpkc47x/AAAHKhwYaI8UkwBR9sx0fw6Fa?dl=0

I have written this program to cover the ground station receiver section.

When you open this program, you will see that the test parameters have been set to some default values.

To use the program, enter the values for your station into the text boxes and press the “Calculate” button.

Notes on the Parameters:

1)     Antenna Noise Figure: This is the background noise figure of the sky at the frequency of interest. I have put a default value of 2dB here which is the generally accepted figure for a suburban environment at 2M. 

For 70CMs, the Antenna Noise figure will be about 1/3 of the Noise figure at 2M.

Set this value of this text box to the Noise Figure for your location and frequency. Leave at 2dB if unsure.

This reference paper gives a set of test measurements for various frequencies. 

https://www.wmo.int/pages/prog/www/TEM/SG-RFC06/Ambient-RF-noise.pdf

 

2)     Coupling/Feeder Loss: This is the insertion loss in dB of the jumper between your antenna and preamp if you are intending to use one at the antenna OR the insertion loss of your main feeder if you are not using a preamp or using an indoor preamp. Use the feeder manufacturers’ data sheet for your cable to obtain this value. I have used a default 0.75dB which is for a 50ft run of LMR-400.

 

3)     Preamp Gain: This is obtained from the manufacturer’s data sheet. I have used the default value of 24dB from

 an ARR  P144VDG preamp. Set to zero if no preamp is used.

 

4)     Preamp Noise Figure: Again this is obtained from the manufacturer’s data sheet. I have used the default value of 0.5dB from an ARR  P144VDG preamp. Set to zero if no preamp is used.

 

5)     Feeder/Coupling Loss: This is the insertion loss in dB of the jumper between your preamp and receiver if a preamp is used indoors OR the insertion loss of your main feeder if you are using a preamp at the antenna.

See the manufacturer’s data sheet for this. I have used a default value of 0.2dB for a short jumper.

 

6)     Receiver Sensitivity: This is usually published in the manufacturer’s data sheet for your radio. This is normally given in microvolts (uV). I have used a default value of 0.25uV which is about right for my Yaesu FT726.

If you have a modern radio e.g. Icom IC9700 then this value is 0.11uV with the internal preamp switched on.

 

7)     Bandwidth: This is bandwidth of the receiver for the mode of interest. I have put this at 3000 Hz for SSB. If you change this, change the Receiver Sensitivity entry to match.

Once you are happy with the inputs, press the “Calculate” to obtain the result.

You can save the results to a text file using the File/Save menu selection.
